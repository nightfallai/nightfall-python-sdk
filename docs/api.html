API¶

Nightfall Object¶

nightfall.api¶

This module provides a class which abstracts the Nightfall REST API.

class nightfall.api.Nightfall(token, condition_set)¶

A python interface for the Nightfall API.

MAX_PAYLOAD_SIZE¶

Maximum payload size that the Nightfall API will accept

MAX_NUM_ITEMS¶

Maximum number of items that the Nightfall API will accept

Instantiate a new Nightfall object.

Parameters

  • token – Your Nightfall API token.

  • condition_set – Your Nightfall Condition Set UUID

make_payloads(data)¶

Turn a list of strings into a list of acceptable payloads.

Creates chunks based on the MAX_PAYLOAD_SIZE and MAX_NUM_ITEMS constants.

Parameters

data – list of string

Returns

list of list of strings

scan(data)¶

Scan lists of data with Nightfall.

This method will convert the list of strings into chunks if necessary and then makes one or more requests to the Nightfall API to scan the data.

Parameters

data (list) – list of strings to scan.

Returns

list of list of resposes for items in payload.
